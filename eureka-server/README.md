## About Eureka Server

Discovery Server with Eureka by Netflix, every microservice will register to this server along with their respective Ports
and IP Address.

<p align="right">(<a href="#top">back to top</a>)</p>



### Built With

The major frameworks/libraries that powered this project.

* [Spring Boot](https://spring.io/projects/spring-boot)
* [Spring Cloud](https://spring.io/projects/spring-cloud)
* [Spring Data JPA](https://spring.io/projects/spring-data-jpa)
* [Spring AMQP](https://spring.io/projects/spring-amqp)
* [RabbitMQ](https://www.rabbitmq.com/)
* [Zipkin](https://zipkin.io/)

<!-- USAGE EXAMPLES -->
## Usage

Resource links (prerequisite: installed locally)

* Discovery Server
    ```
    http://localhost:8761/
    ```
* Postgres Management Tool
    ```
    http://localhost:5050/browser/
    ```
* Distributed Tracing Tool
    ```
    http://localhost:9411/zipkin/
    ```
* Message Queue Management Tool
    ```
    http://localhost:15672/
    ```
